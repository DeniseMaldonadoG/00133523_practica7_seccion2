¿Cuál es la diferencia entre autenticación y autorizacion?
La autenticación es el proceso de verificar la identidad de un usuario (confirmar que es quien dice ser), mientras que la autorización es el proceso de determinar qué permisos tiene ese usuario para acceder a recursos específicos una vez que se ha autenticado.

¿Cuál es la función del token JWT en la guía?
s una forma segura de representar la identidad y permisos del usuario después de autenticarse.
